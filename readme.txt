説明

本プロジェクトは、産業用コンピュータ試験において、最大構成試験（Fully Equipped robustness Test）と連続ON/OFF試験（Continuous On/Off robustness Test）の一部自動化を図るツールキットです。


フォルダー構造

common　　 -　共有ライブラリ（作成中）

cot-host   -　ON/OFF試験のホスト側プログラム　　　（作成中）

cot-target -　ON/OFF試験のターゲット側プログラム　（作成予定）

fet-host   -　最大構成試験のホスト側プログラム　　（作成予定）

fet-target -　最大構成試験のホスト側プログラム　　（作成予定）
